# Yummy Food Restaurant 🍽️

一個可在電腦、手機及平板瀏覽器直接遊玩的 3D 教育遊戲。

玩家走進動畫電影感餐廳後，可自由選擇健康或不健康食物，並即時觀察角色身體變化。

- 🥦 健康食物：增加健康與力量、降低體脂
- 🍔 高糖高脂食物：增加體脂、降低健康
- 💪 力量增加時，角色手臂會逐漸變強壯
- 🍩 體脂增加時，角色身體和肚子會逐漸變胖

## 最新角色畫風

角色已重做成高質 3D 動畫電影／寫實卡通渲染風格，包括：

- 大眼睛、虹膜、高光、眉毛、鼻子、笑容和紅潤面頰
- 分層立體頭髮與柔和材質
- 紅色連帽外套、深色內衣、長褲及運動鞋
- 走路、手腳擺動、呼吸、眨眼和吃東西動畫
- 電影感主光、輪廓光、暖色餐廳燈光及柔和陰影

## 遊戲功能

- 360° 自由旋轉第三身鏡頭
- 鏡頭縮放及一鍵回到角色背後
- 移動方向會跟隨當前鏡頭方向
- 8 種可互動食物
- 健康、力量、體脂及分數系統
- 即時角色體型變化
- 桌椅及櫃台碰撞系統
- 支援 iPhone、Android、iPad 及電腦
- 不需要安裝或編譯

## 操作方法

### 電腦

- `WASD` 或方向鍵：移動
- 拖動滑鼠：360° 旋轉視角
- 滑鼠滾輪：拉近或拉遠鏡頭
- `E` 或空白鍵：吃食物
- `↻`：把鏡頭重設到角色背後

### 手機／平板

- 左下角搖桿：移動
- 拖動遊戲畫面：360° 旋轉視角
- 右下角「食」按鈕：吃食物
- 建議橫向遊玩

## 遊戲目標

把力量提升至 80，同時把體脂保持在 40 以下。

## GitHub Pages

1. 開啟 Repository 的 `Settings`
2. 進入 `Pages`
3. 在 `Build and deployment` 選擇 `Deploy from a branch`
4. Branch 選擇 `main`，資料夾選擇 `/ (root)`
5. 儲存後等待 GitHub Pages 產生網址
